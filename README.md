구매 물품 관리 앱

이 웹앱은 식재료, 생활용품 등 다양한 구매 물품을 카테고리별로 관리하고, 필요한 수량을 체크하며, 선택한 물품만 손쉽게 공유할 수 있도록 도와줍니다.

주요 기능
	•	카테고리별 물품 리스트/필요 수량 관리
	•	체크박스 선택 → 대시보드에서 한눈에 확인
	•	선택 항목만 메시지/클립보드로 손쉽게 공유
	•	새 물품 추가, 수량/단위/카테고리 수정 가능
	•	모바일 최적화 및 스와이프(카테고리 이동) 지원

사용법
	1.	상단에서 원하는 카테고리를 선택하거나 좌우로 스와이프합니다.
	2.	물품을 체크하면 대시보드(상단)에 표시됩니다.
	3.	대시보드의 “전체 해제” 버튼으로 선택 해제 가능.
	4.	“공유” 아이콘 클릭 → 체크된 물품만 공유(복사/메시지)
	5.	“새 항목 추가” 폼에서 직접 물품을 추가할 수 있습니다.

Purchase Inventory Manager App

This web app helps you manage your shopping inventory by category, keep track of required quantities, and easily share your selected items with others.

Features
	•	Manage items and quantities by category
	•	Checkbox selection with real-time dashboard overview
	•	Easily share only the selected items via message or clipboard
	•	Add new items; edit quantity, unit, and category
	•	Mobile-friendly with swipe support for category navigation

How to Use
	1.	Select a category at the top or swipe left/right.
	2.	Check items to add them to the dashboard at the top.
	3.	Use the “Clear All” button on the dashboard to uncheck all.
	4.	Click the “Share” icon to share only the checked items.
	5.	Add new items through the “Add New Item” form.

Einkaufsliste Management App

Mit dieser Web-App können Sie Ihre Einkaufsartikel nach Kategorien verwalten, benötigte Mengen verfolgen und ausgewählte Artikel ganz einfach teilen.

Hauptfunktionen
	•	Verwaltung von Artikeln und Mengen nach Kategorie
	•	Auswahl per Checkbox, Überblick im Dashboard
	•	Nur ausgewählte Artikel einfach per Nachricht oder Zwischenablage teilen
	•	Neue Artikel hinzufügen, Menge/Einheit/Kategorie bearbeiten
	•	Für Mobilgeräte optimiert, Swipe-Unterstützung für Kategorien

Anleitung
	1.	Oben eine Kategorie auswählen oder nach links/rechts wischen.
	2.	Gewünschte Artikel ankreuzen – sie erscheinen im Dashboard.
	3.	Im Dashboard mit „Alle löschen“ Auswahl aufheben.
	4.	Über das „Teilen“-Symbol nur ausgewählte Artikel weitergeben.
	5.	Neue Artikel über das Formular „Neuen Artikel hinzufügen“ anlegen.
